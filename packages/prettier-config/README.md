# Description

Predefined prettier configuration.

# Install

```bash
npm i @kbotdev/prettier-config -D
```

# Usage

```js
// .prettierrc.js

module.exports = require('@kbotdev/prettier-config');
```

or

```json
// package.json

{
	"prettier": "@kbotdev/prettier-config"
}
```
